# Unit Turn Priority Dashboard

A prioritized task list for property managers to track all steps in turning a unit — from non-renewal through collections.

## Story-Driven Delivery Demo

This project demonstrates the [Story-Driven Delivery](https://github.com/RealPage/agentic-workflows/blob/main/docs/workflows/story-driven-delivery.md) workflow — an end-to-end agentic SDLC process where teams build deep understanding of the customer problem before agents start building.
